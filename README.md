# 🌐 Faria Bente Faruk – Personal Portfolio Website

This is a personal portfolio website built using **HTML5 and CSS3**.  
It showcases my education, technical skills, and academic projects as a CSE undergraduate student.

---

## 📌 Project Overview

This portfolio website contains:

- Fixed sidebar with profile image  
- About Me section  
- Education history  
- Skills with progress bars  
- Project showcase cards  
- Contact information  

The design uses a lavender-themed modern UI with a clean layout.
